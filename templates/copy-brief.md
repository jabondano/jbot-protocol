## Messaging Hierarchy

### Headline Options (pick one before generating creative)

| Option | Copy | Angle |
|--------|------|-------|
| A | `[headline]` | Contrarian / "Don't Buy" |
| B | `[headline]` | Benefit-forward |
| C | `[headline]` | Price / Value |

Scoring guide:
- Contrarian hooks tend to win with AI-Forward Early Adopters (13.2x ROAS precedent)
- Benefit-forward wins with Active Lifestyle Consumers (5.0x ROAS precedent)
- Price hooks work with Prescription Eyewear Upgraders ("priced lower than you think")

### Body Copy (2 lines max)

```
Line 1: [Pain/situation the ICP recognizes]
Line 2: [Product as the natural solution — no technical specs, just outcome]
```

### CTA

```
[Single action. One of: Shop Now / See It In Action / Get Yours / Try Them On]
URL: [product page or collection page]
```

## Visual Brief

```
Product image:  [Shopify CDN Perspective.jpg URL — mandatory]
Motion:         [image-to-video via Seedance 2.0] OR [static composite via ffmpeg]
Key visual:     [what the viewer sees in the first frame — the hook]
Product moment: [how the product appears — close-up, worn, on surface]
USP delivery:   [behavior, not interface — show what you DO, not what you SEE]
```

**Reminder:** Open ears = the USP for audio glasses. Show someone in the world, not isolated.

---

## Approval Checklist

Before approving this brief for creative execution:

- [ ] Copy is specific to this ICP — not generic
- [ ] Headline can stand alone without seeing the product
- [ ] Body is 2 lines or fewer
- [ ] CTA is one action only
- [ ] Product image URL confirmed (Shopify CDN, not local file)
- [ ] No AR/lens/HUD planned
- [ ] No earbuds/headphones in the scene
