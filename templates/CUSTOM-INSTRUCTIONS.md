## Template Notes (Do Not Include Above)

### Tips for Effective Custom Instructions

1. **Be specific** — Vague instructions lead to inconsistent behavior
2. **Include examples** — Show don't just tell
3. **Define boundaries** — Clear limits prevent overreach
4. **Update regularly** — Instructions should evolve with the role
5. **Test thoroughly** — Verify the agent behaves as expected

### Testing Checklist

- [ ] Agent correctly handles standard requests
- [ ] Agent appropriately escalates edge cases
- [ ] Agent respects defined boundaries
- [ ] Agent uses correct tone and formatting
- [ ] Agent properly uses available tools
