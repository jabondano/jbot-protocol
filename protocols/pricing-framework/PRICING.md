# JBOT Protocol Pricing Framework

## The Four Value Dimensions

### 1. Direct Cost Savings
What manual work does the fleet replace? Calculate FTE-equivalent hours × loaded cost.

### 2. Token & Infrastructure Cost (pass-through)
Actual API costs + 20% margin. Never bundle — always visible line item.

### 3. Opportunity Cost (Most powerful sales argument)
Every month a company delays, competitors compound intelligence. Formula:

```
Monthly Competitive Lag Cost = 
  (Decisions automated/mo × Avg decision value)
  + (Analyst hours saved/mo × $75)
  + (Conversion rate delta from ICP intelligence × monthly revenue)
```

Target: Show >$20K/month lag cost for a $3-5M company. Deployment pays for itself in 4-8 weeks on opportunity cost alone.

### 4. Accrued Knowledge Value
The system gets smarter every month. ICP profiles, B2B signal data, hypothesis backlogs, Supabase schemas — this is compounding institutional intelligence. After 12 months it's nearly irreplaceable.

Knowledge handoff fee if client exits: 3× monthly retainer.

## Build Fee Formula

```
Build Fee = $2,500 × Process Count × Complexity Multiplier × Division Multiplier × [Urgency Premium]

Complexity:
  Simple  (1-2 integrations, existing data):      1.0x
  Medium  (3-5 integrations, custom schema):       1.5x
  Complex (6+ integrations, real-time, multi-DB):  2.0x

Division Multiplier = 1 + (0.25 × divisions beyond first)

Urgency Premium = +15-25% if competitive lag is active
```

### Reference Points
| Company Type | Processes | Complexity | Divisions | Est. Build Fee |
|-------------|-----------|-----------|-----------|---------------|
| Small ops (3 bots) | 3 | Simple | 1 | ~$7,500 |
| Mid-market (5 bots) | 5 | Medium | 1–2 | ~$18,750–$23,000 |
| Full enterprise fleet | 10+ | Complex | 3–5 | ~$50,000–$80,000 |

## Monthly Retainer Formula

```
Retainer = (Bot Count × $200) + (Active Crons × $50) + (Token Cost × 1.2) + $500 base

Year 2+ escalator: +10%/year (system compounds in value)
```

---

## ROI Qualifier

Only pursue engagements where:
```
Annual Savings ÷ Year 1 Cost > 2.5x
```

If ROI is below 2.5×, the client either needs a smaller scope or isn't ready.

---

## Project Evaluation Checklist (5 Questions Before Quoting)

1. How much domain knowledge/data do they already have?
2. How many manual workflows exist that can be automated?
3. What's their tolerance for AI errors? (Low = higher testing hours)
4. Do they have a technical contact? (No = higher maintenance retainer)
5. What's their competitive lag? (Behind = urgency premium justified)
