# Copilot Pattern

Single-purpose internal tools, built ad hoc the same week the need is discovered, governed proportionally.

**The problem it solves:** every operation carries a long tail of small problems that were never worth months of dev time, a SaaS subscription, or a consulting engagement. AI-assisted building flipped that math — but ungoverned ad hoc building produces orphaned-script sprawl.

**The pattern in one line:** deterministic core + AI edges, a named output owner, a one-page manual, and a slot in the fleet's recurring loop.
