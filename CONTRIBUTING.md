# Contributing to JBOT Protocol

Thank you for your interest in improving the JBOT Protocol.

## How to Contribute

### Reporting Issues

- **Errors or broken links:** Open a [Bug Report](https://github.com/jabondano/jbot-protocol/issues/new?template=bug-report.md)
- **Improvement suggestions:** Open an [Improvement](https://github.com/jabondano/jbot-protocol/issues/new?template=improvement.md) issue
- **Questions:** Use [GitHub Discussions](https://github.com/jabondano/jbot-protocol/discussions)

### Suggesting Changes

1. Open an issue describing the change you'd like to see
2. Reference specific files and sections
3. Explain the reasoning behind the suggestion

### What We're Looking For

- Corrections to factual errors
- Improvements to clarity and readability
- Additional examples or case studies (anonymized)
- MCP ecosystem updates (new servers, tools)
- Model capability updates (pricing, features)

### What We're Not Looking For

- Changes to the core framework methodology
- Promotional content or tool-specific endorsements
- Content that conflicts with the framework's principles
